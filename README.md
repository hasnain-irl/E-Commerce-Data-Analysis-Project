Overview:
This project demonstrates an end-to-end data analysis workflow using a simulated E-commerce dataset. The objective was to clean raw data, perform analysis, and generate business insights through visualization.

Tools Used:
MySQL (Data Cleaning & Analysis)
Excel (Data Preprocessing)
Power BI (Dashboard & Visualization)

Dataset:
The dataset was generated using Mockaroo and includes:
500 Customers
20 Products
1000 Orders

The data contains real-world issues such as:
Missing values
Invalid formats
Duplicate records
Inconsistent text formatting

Data Cleaning:
Performed using SQL and Excel:

Removed duplicates
Handled NULL values
Standardized city names
Fixed formatting issues
Validated email formats

Data Analysis:
Key SQL operations performed:

JOIN operations across multiple tables
Aggregations and Window functions
Grouping and filtering
Update and Delete

Key Insights:
Top customers contribute a significant portion of total revenue
Certain products dominate overall sales
Sales vary significantly across months (seasonal trends)
Revenue distribution differs by city

Dashboard
The Power BI dashboard includes:

Sales Trend Over Time
Revenue by City
Top Products
Customer Spending
Revenue by City
Top Products
Customer Spending
